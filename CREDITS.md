### Descripción del Original
Demo de césped estilizado con animación de viento implementado en Three.js vanilla JavaScript.

## Migración a React + TypeScript

**Adaptado por**: [Tu Nombre]  
**Fecha**: Noviembre 2025  
**Objetivo**: Migración exacta a React + TypeScript manteniendo funcionalidad y apariencia visual idénticas

### Cambios Realizados
- Migración a React + TypeScript
- Uso de React Three Fiber como abstracción de Three.js
- Organización en componentes React
- Tipado estático con TypeScript
- Build system actualizado a Vite
- Three.js actualizado a v0.159.0 (por compatibilidad con dependencias modernas)

### Código Mantenido Sin Cambios
- Shaders GLSL (vertex y fragment) - Copia exacta
- Algoritmo de generación de geometría - Lógica idéntica
- Parámetros de configuración - Valores exactos
- Distribución de blades - Mismo algoritmo
- Sistema de animación - Mismos cálculos
